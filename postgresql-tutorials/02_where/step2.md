## Where 句

- "FROM" -> "WHERE" -> "SELECT" -> "ORDER BY"


| 演算子     | 詳細                                                           |
| :--------: | :------------------------------------------------------------- |
| =          | 等しい                                                         |
| >          | より大きい                                                     |
| <          | 未満                                                           |
| >=         | 以上                                                           |
| <=         | 以下                                                           |
| <> , !=    | と等しくない                                                   |
| AND        | 論理演算子としての AND                                         |
| OR         | 論理演算子としての OR                                          |
| IN         | 与えられた値がリストにある場合に true を返す                   |
| BETWEEN    | 与えられた値が, 指定した値と値の間にある場合に true を返す     |
| LIKE       | 与えられた値が, 指定したパターンにマッチした場合に true を返す |
| IS NULL    | 与えられた値が, NULL の場合に true を返す                      |
| NOT        | 他の演算子の結果を否定する                                     |
